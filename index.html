How to add new compounds

You don't need to understand code to add entries. Just find the data section near the bottom of this file, copy a template, fill in your values, and save. That's it.

1

Open index.html in a text editor

On Mac: use TextEdit (switch to plain text mode first via Format → Make Plain Text). On Windows: use Notepad. Or use VS Code if you have it — it's free and highlights the code nicely.

2

Find the molecule data section

Search for the text const molecules = [ — that's where all compounds live. Each entry starts with {id: and ends with },

3

Copy the right template and fill it in

Use the chromophore template for fluorophores/photosensitizers, the radical template for EPR-active species, or the dyad template for covalently linked systems.

4

Paste before the closing bracket and save

Add your new entry right before the ]; line that closes the array. Make sure the previous entry ends with a comma. Save the file and re-upload it to GitHub.
